# GitHub-Finder

Application web simple et moderne pour rechercher des profils GitHub, afficher leurs informations et sauvegarder des favoris.

##  Fonctionnalités

- 🔍 Rechercher un utilisateur GitHub
-  Afficher les informations du profil :
  - Nom
  - Bio
  - Localisation
  - Followers / Following
  - Nombre de repos
- 📦 Voir les 5 repositories les plus populaires
- ⭐ Ajouter / supprimer des favoris
- 💾 Sauvegarde des favoris avec localStorage
- ⚡ Interface rapide et responsive

##  Technologies utilisées

- HTML5
- CSS3 (Design moderne + animations)
- JavaScript (Vanilla JS)
- GitHub API
